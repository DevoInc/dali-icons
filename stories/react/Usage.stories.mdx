# Usage

```jsx
import { DvCheck } from "@devoinc/dali-icons";

export const Example = () => (
  <div>
    <DvCheck size={64} color={'rgba(0,0,180,1)'} />
  </div>
);
```

## Props

- **size:** the size of the icon
- **color:** the color of the icon
- **title:** the internal title of the icon, [more info...](https://developer.mozilla.org/en-US/docs/Web/SVG/Element/title)
- **className:** a class name for css or link
- **style:** custom styles for the svg element

For see all the props in action go to the [icon story](?path=/story/react-icon--single-icon) :)

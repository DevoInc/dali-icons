# Usage

With this you will have all icons classes:

- `.gi-check`
- `.gi-delete`
- `.gi-refresh`
- ...

You could check all the classes in the [Gallery](?path=/story/gallery--icons)

## CSS

```js
import '@devoinc/genesys-icons/dist/gi-styles.css';
```

## SASS

```js
import '@devoinc/genesys-icons/dist/gi.scss';
```

## LESS

```js
import '@devoinc/genesys-icons/dist/gi-styles.less';
```
